Sobre Mi

Soy Analista de Datos de Fútbol, combinando una sólida formación en Ingeniería Matemática con un Máster de Formación Permanente en Big Data aplicado al Fútbol por la prestigiosa Escuela Universitaria Real Madrid Universidad Europea. Mi misión es traducir datos complejos en inteligencia táctica clara y accionable para apoyar a entrenadores, analistas y directivos.

Mi proyecto de fin de máster, Match Explorer, es una plataforma interactiva que desarrollé desde cero. Transforma datos de eventos de partidos en visualizaciones avanzadas, ayudando a los cuerpos técnicos a analizar fases del juego (construcción de juego, transiciones ofensivas y defensivas, y jugadas a balón parado). El objetivo es descubrir patrones tácticos en los rivales y obtener una ventaja competitiva.

Mi experiencia abarca el ciclo de vida completo del dato. Construyo pipelines de datos automatizados con Python y R (Pandas, rvest, dplyr), garantizo la integridad de los datos con esquemas SQL robustos y transformo hallazgos complejos en historias visuales impactantes, desde aplicaciones personalizadas en Dash/Plotly hasta plataformas de BI como Tableau.

2025

Match Explorer: Plataforma de Análisis Táctico

Como proyecto central de mi Máster en la Escuela Universitaria Real Madrid Universidad Europea, desarrollé Match Explorer, una aplicación web completa construida desde cero con Python, Dash y Plotly. El proyecto simula el flujo de trabajo de un analista profesional, encargado de apoyar al cuerpo técnico del Liverpool FC transformando datos de eventos en insights tácticos accionables.

La plataforma ofrece un análisis profundo de las dinámicas del juego, permitiendo un estudio detallado de fases tácticas clave como la construcción de juego, transiciones ofensivas y defensivas, y jugadas a balón parado. Incluye visualizaciones interactivas como redes de pases, mapas de pases progresivos y métricas avanzadas como el PPDA (Passes Per Defensive Action) para evaluar la intensidad de la presión. El objetivo final es identificar patrones en los rivales y validar planes de partido, como se demostró en el análisis pre y post-partido para los encuentros clave de abril del Liverpool.

2024

Seguimiento de Jugadores y Balón con YOLOv8

Este proyecto implementa un pipeline de visión por computador para detectar y seguir jugadores, árbitros y el balón en vídeos de partidos. Entrené un modelo YOLOv8 personalizado sobre un dataset de Roboflow y utilicé ByteTrack para un seguimiento consistente de los objetos. Una funcionalidad clave es la asignación automática de equipos, que usa un algoritmo de clustering KMeans para diferenciar equipos según el color de sus camisetas.

2024

Reportes de Fútbol Automatizados con R

Un pipeline de datos end-to-end y automatizado, construido en R, que extrae diariamente estadísticas de fútbol de FBref usando rvest, procesa los datos con dplyr y genera informes dinámicos en HTML con rmarkdown. Todo el flujo de trabajo está programado para ejecutarse diariamente con taskscheduleR, garantizando que los insights estén siempre actualizados.

  • Proyecto en Github
2024

Base de Datos SQL para Gestión de Lesiones

Este proyecto demuestra competencias clave en ingeniería de datos a través de una base de datos relacional construida con SQL. Diseñada desde cero, gestiona datos de lesiones de futbolistas, incluyendo tratamientos y especialistas involucrados. El esquema está normalizado para garantizar la integridad de los datos y soporta consultas complejas para analizar patrones de lesiones, eficacia de tratamientos y carga de trabajo de los especialistas.

2023

Dashboard de COVID-19 con Tableau

Este proyecto ilustra un flujo de trabajo completo de análisis de datos. Procesé y consulté un gran dataset sobre COVID-19 de Our World in Data usando SQL para extraer métricas clave sobre tasas de infección y mortalidad. Los resultados fueron visualizados en un dashboard público e interactivo creado con Tableau para hacer los datos complejos más claros y accesibles.
